# 🤖 Local AI Chat — чат с документами на локальной модели

Загружаешь PDF/Markdown → задаёшь вопросы → модель отвечает только по твоим документам.
Всё работает локально, без интернета, без OpenAI.

---

## 🛠 Установка

### 1. Установить Ollama

**macOS / Linux:**
```bash
curl -fsSL https://ollama.com/install.sh | sh
```

**Windows:**
Скачай установщик с https://ollama.com/download

---

### 2. Скачать языковую модель

После установки Ollama открой терминал и выполни:

```bash
# Лёгкая и быстрая (рекомендуется для начала):
ollama pull mistral        # ~4 GB, отлично читает документы

# Другие варианты:
ollama pull llama3.2       # ~2 GB, быстрее, чуть слабее
ollama pull gemma2         # ~5 GB, хорош для рассуждений
ollama pull phi4           # ~9 GB, лучшее качество
```

Скачивание займёт несколько минут (зависит от интернета).

---

### 3. Запустить Ollama

```bash
ollama serve
```

Оставь этот терминал открытым — Ollama должна работать в фоне.

---

### 4. Клонировать / распаковать проект

```bash
cd local-ai-chat
```

---

### 5. Создать виртуальное окружение и установить зависимости

```bash
python -m venv venv

# Linux / macOS:
source venv/bin/activate

# Windows:
venv\Scripts\activate

pip install -r requirements.txt
```

---

### 6. Запустить приложение

```bash
uvicorn app:app --reload --port 8000
```

Открой браузер: **http://localhost:8000**

---

## 🎮 Как пользоваться

1. **Загрузи документ** — перетащи PDF или Markdown в левую панель
2. **Выбери документы** — кликни на них (подсветятся фиолетовым)
3. **Задай вопрос** — введи в поле снизу и нажми Enter

---

## ⚙️ Смена модели

Открой `app.py` и измени строку:

```python
OLLAMA_MODEL = "mistral"   # ← поставь имя любой скачанной модели
```

Список скачанных моделей: `ollama list`

## 🔍 Как работает поиск

1. Документ разбивается на перекрывающиеся чанки по ~1500 символов
2. При вопросе ищутся чанки с наибольшим совпадением слов
3. Топ-5 чанков отправляются в Ollama как контекст
4. Модель отвечает строго по контексту

> Это простой keyword-поиск. Для больших документов можно улучшить до векторного поиска (FAISS + sentence-transformers).

## 🚨 Частые проблемы

| Проблема | Решение |
|---|---|
| `ollama не запущена` | Выполни `ollama serve` в терминале |
| Модель не отвечает | Проверь `ollama list` — модель должна быть скачана |
| Медленно | Используй меньшую модель (`llama3.2`, `phi3-mini`) |
| Ошибка порта | Измени порт: `uvicorn app:app --port 8080` |
